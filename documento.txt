# 📘 CURSO PHP MODERNO

---

## 2. Evolução do PHP
- **1994** – Criado por *Rasmus Lerdorf*  
- **1995** – Versão 1: *PHP Tools* (form interpreter)  
- **1996** – *PHP/FI*: integração com banco de dados, código aberto  
- **1997** – *Andi Gutmans* e *Zeev Suraski* tornam-se grandes colaboradores  
- **1998** – PHP começa a se consolidar como linguagem  
- **1998** – *Colin Viebrock* e *Vincent Pontier* criam o mascote do PHP  
- **1999** – Surge a empresa *Zend*, para contribuir com o PHP  

---

## 3. O mascote do PHP 🐘
- Nome: **ElePHPant**  
- Origem: "PHP" lembra a forma de um **elefante**  
- Significado:  
  - Antes: *Personal Home Page*  
  - Atual: *PHP Hypertext Preprocessor*  

---

## 4. Principais versões do PHP
| Ano | Versão | Destaque |
|-----|--------|----------|
| 1995 | PHP 1 | Contador simples |
| 1997 | PHP/FI 2.0 | Nome oficial |
| 1998 | PHP 3.0 | Linguagem de programação |

| 2004 | PHP 5.0 | Orientação a objetos |

## 6. Como funciona o PHP
- **Client-side**: navegador do usuário  
- **Server-side**: servidor processa e retorna resposta  
- Fluxo: *Request → Servidor → Response*  

---

## 7. Configurando ambiente de desenvolvimento
- **Servidor local**:  
  - Linux → LAMP  
  - Windows → WAMP  
  - Mac → MAMP  
  - Multiplataforma → XAMPP  
- **Editor recomendado**: VSCode  
  - Extensões úteis:  
    - Portuguese  
    - PHP Intelephense  
    - PHP Awesome Snippets  

---

## 8. Sintaxe do PHP
- **Super tag**: `<?php ?>`  
- **Short tag**: `<? ?>`  
- **Echo simplificado**: `<?= ?>`  

---

## 9. Variáveis e constantes
- Variáveis: começam com `$`  
- Constantes: valores fixos, não mudam  
- Armazenamento: memória do servidor  

---

## 10. Tipos de dados
- **Escalares**: string, int, float, boolean  
- **Compostos**: array, object  
- **Especiais**: null, resource, mixed  

---

## 11. Strings e concatenação
- Aspas duplas `"texto"` → interpretam variáveis  
- Aspas simples `'texto'` → não interpretam  
- Concatenação: `"Curso" . "PHP"`  

---

## 12. Operadores e precedência
- **Potência**: `**`  
- **Multiplicação/Divisão**: `* / %`  
- **Adição/Subtração**: `+ -`  
- Uso de parênteses altera ordem  

---

## 13. Funções matemáticas úteis
- `abs()` → valor absoluto  
- `ceil()`, `floor()`, `round()` → arredondamentos  
- `sqrt()` → raiz quadrada  
- `pow()` → potência  
- `sin()`, `cos()`, `tan()` → trigonometria  

---

## 14. Superglobais do PHP
- `$_GET`, `$_POST`, `$_REQUEST`  
- `$_FILES`, `$_SESSION`, `$_COOKIE`  
- `$_ENV`, `$_SERVER`, `$GLOBALS`  

---

## 15. Segurança básica
- Cuidado com **query strings**  
- Validação de dados enviados via `POST`  
- Proteção contra injeções e manipulação de variáveis
